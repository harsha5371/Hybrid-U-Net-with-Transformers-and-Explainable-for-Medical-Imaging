# Hybrid-U-Net-with-Transformers-and-Explainable-for-Medical-Imaging
Built a deep learning model for brain tumor segmentation using U-Net with VGG19 and Vision Transformer. Trained on BraTS 2020 MRI data. Used Grad-CAM for explainability to visualize which parts of the image influenced the model’s predictions, making the results more interpretable and transparent.
